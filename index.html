How to Import into Figma

Step 1: Download the SVG

First, download the SVG file by right-clicking on the image above and selecting "Save Image As..." or by clicking the button below.

Download SVG

Step 2: Import into Figma

Open Figma and create a new file or open an existing one. Then:

  1. Click on "File" in the top menu
  2. Select "Import..." or press Shift+Cmd+I (Mac) / Shift+Ctrl+I (Windows)
  3. Choose the downloaded SVG file
  4. The card will be imported as a vector that you can edit

Step 3: Customize the Card

In Figma, you can now:

  • Edit any text, colors, or elements
  • Resize or reposition components
  • Add your own logo or branding
  • Create variants for different account types
  • Add animations or interactions

Direct SVG Code

If you prefer, you can also copy the SVG code directly and paste it into Figma as a code component:

  1. View the SVG file on GitHub
  2. Copy the raw SVG code
  3. In Figma, paste using Paste as SVG option
